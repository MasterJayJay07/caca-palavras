Caça palavras
Regras do jogo:
  1. As palavras estão escondidas horizontalmente dentro da grade.
  2. você deve encontrar 3 palavras dentro do caça palavras
  3. As palavras são arranjadas normalmente de modo que possam ser lidas da esquerda para a direita.
  4. Basta clicar nas letras para formar as palavras, se as 3 palavras estiverem corretas você vence.
  5. As possiveis palavras a serem encontradas são: 'PIKACHU', 'TAUROS', 'CHARIZARD', 'DRAGONITE', 'ONIX', 'ARBOK', 'RAICHU', 'KADABRA', 'VULPIX', 'ARCANINE', 'ZUBAT', 'HORSEA', 'SNORLAX', 'ARTICUNO', 'LAPRAS', 'MAGIKARP', 'MOLTRES', 'ZAPDOS', 'MEWTWO', 'LUCARIO'
